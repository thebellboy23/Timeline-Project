# Timeline-Project
A local web-app to create notes on a timeline and search for them. (A lot of the code was revised by Claude AI)
## Usage
Open `index.html` in browser to run the app.
##
### What is is about
This timeline enables you to make notes on a timeline, zoom to specify dates, rename notes, search for notes or dates and a few other functions. It has minor issues but it's open source for whoever wasnts to use it. It can be very helpful for history students or that purpose, or it can be edited in source code to make a study plan timline.
